# Hassan Portfolio

A modern and responsive portfolio website built with React and Tailwind CSS. This portfolio showcases my skills, projects, and professional experience in an elegant and interactive way.

## Features

- 🎨 Modern and clean design
- 📱 Fully responsive layout
- ⚡ Fast and optimized performance
- 🌙 Dark/Light mode support
- 🎯 Interactive animations
- 📝 Project showcase
- 👨‍💻 Skills section
- 📞 Contact form
- 🌐 Multi-language support

## Tech Stack

- React.js
- Tailwind CSS
- Framer Motion
- React Icons
- React Country Flag

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository
```bash
git clone https://github.com/Aqeelbmh/Hassan_Portfolio.git
```

2. Navigate to the project directory
```bash
cd Hassan_Portfolio
```

3. Install dependencies
```bash
npm install
# or
yarn install
```

4. Start the development server
```bash
npm run dev
# or
yarn dev
```

5. Open your browser and visit `http://localhost:3000`
